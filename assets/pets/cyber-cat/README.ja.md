# サイバーキャット

このフォルダーには、デフォルトの無料ピクセルペットパックが含まれています。

スプライトシートは、このリポジトリのために生成されたオリジナルアートであり、リポジトリの0BSDライセンスの下で公開されています。後で手作成のAsepriteアセットに置き換えられるように、意図的に小さくシンプルに作られています。

推奨アセット構成:

```txt
cyber-cat/
  manifest.json
  spritesheet.png
  spritesheet.json
```

デフォルトのプレースホルダーアートを再生成するには、次のコマンドを実行します:

```bash
npm run assets:generate
```
