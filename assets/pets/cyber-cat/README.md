# cyber-cat

This folder contains the default free pixel pet pack.

The sprite sheet is original art generated for this repository and is released
under the repository 0BSD License. It is intentionally small and simple so it can
be replaced later with hand-authored Aseprite assets.

Recommended asset shape:

```txt
cyber-cat/
  manifest.json
  spritesheet.png
  spritesheet.json
```

Regenerate the default placeholder art with:

```bash
npm run assets:generate
```
