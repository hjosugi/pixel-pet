# kofun-friend

このフォルダーには `kofun-friend` ペットパックが含まれています。

現在のスプライトシートはこのリポジトリのために生成され、リポジトリの 0BSD ライセンスの下でリリースされています。

プロジェクトの方向性：Kofun Friends のアートは、実際のソースファイルが追加されるまで、ここでそのまま使用できます。それまでは、生成されたプレースホルダーがそのまま残ります。

デフォルトのプレースホルダーアートを再生成するには、次のコマンドを実行してください：

```bash
npm run assets:generate
```
