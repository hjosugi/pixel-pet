# サイバー・ペンギン

このフォルダーには、オリジナルの無料ピクセルペットパックが含まれています。

スプライトシートはこのリポジトリのために生成され、リポジトリの0BSDライセンスの下でリリースされています。これは、かわいいサイバーパンクペンギンペットのプレースホルダーパックであり、後で手作成のアートに置き換えることができます。

デフォルトのプレースホルダーアートを再生成するには、次のコマンドを実行してください：

```bash
npm run assets:generate
```
